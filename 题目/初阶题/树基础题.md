#### 三、一棵哈夫曼树共有215个结点,对其进行哈夫曼编码,共能得到()个不同的码字--108
哈夫曼树并不是满二叉树，是正则二叉树（也叫正规二叉树），即其中只有度为0和度为2的结点
因为n0 = n2 + 1，n = n0 + n2; 所以 n = 2n0 - 1，即n0 = (n + 1) / 2;叶子结点n0对应的即是不同的编码。
至于满二叉树当然也是正则二叉树的特例。

哈夫曼压缩的是编码长度：压缩的前提是要知道压缩对象出现的编码类型和各自的出现频率
出现频率就是权值的实际意义
这样才能够构建哈夫曼树，进行不等长编码
